# zh-hans-archive

## 项目简介

这是一个用于归档和发布多个独立 HTML 页面的仓库。本仓库不针对任何特定域名或网站，仅作为静态 HTML 文档的集合存储与展示空间。所有页面内容均为独立编写或整理，适用于信息存档、示例展示或轻量级文档发布等用途。

## 目录说明

```
.
├── index.html         # 仓库首页 / 页面索引
├── pages/             # 存放各独立 HTML 页面的目录
│   ├── example-1.html
│   ├── example-2.html
│   └── ...
├── assets/            # 公共资源文件（CSS、JS、图片等）
│   ├── style.css
│   └── script.js
└── README.md          # 本文件
```

- `pages/`：所有独立归档的 HTML 页面存放于此，每个文件为一个独立页面。
- `assets/`：存放多个页面可能共用的样式、脚本或图片资源。
- `index.html`：提供页面索引或导航，便于浏览仓库内容。

## 页面归档说明

- 每个 HTML 页面均为独立文件，可单独访问或部署。
- 页面内容可能涉及不同主题或用途，彼此之间无直接关联。
- 本仓库不收集、存储或传输任何用户数据，仅作为静态文件托管。
- 所有页面均为纯前端实现，不依赖后端服务。
